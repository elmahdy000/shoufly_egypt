# 🎯 Shoofly: Ultra-Consistent Premium RTL Design Brief

## 1. The Harmonic Rule (Consistency First)
- **Constraint:** Every single UI element must share the same design DNA. No deviations in border-width, icon-stroke, or font-weights.
- **Visual Style:** **Clean High-End Tech**. 
- **Icons:** **Lucide-Style Only**. Stroke width: `1.5px`. Color: `Same as text` (muted) or `Primary` (active). No filled icons mixed with outlined ones.

## 2. Strict Color Strategy
- **Base Background:** `#F4F7FF` (Very light blue-tinted gray).
- **Surface (Cards/Modals):** `#FFFFFF` (Pure white).
- **Primary Accent:** `#0052FF` (Electric Blue) - **Only** for primary buttons and active states.
- **Text Primary:** `#1A1C1E` (Off-black).
- **Text Muted:** `#636467` (Deep gray).
- **Borders:** `1px solid #E2E2E5` (Very subtle borders, NO thick brutalist borders this time).

## 3. Typography & Spacing
- **Font:** `Tajawal` (Arabic) / `Plus Jakarta Sans` (Numbers).
- **Grids:** Strict `8px` spacing system.
- **Rounding:** Uniform `12px` for all cards and buttons.

## 4. Screen Implementation: Admin Dashboard (Arabic RTL)
- **SideNavBar (Right):** Thin, elegant, white background, no background for inactive items. Active item gets a small vertical blue bar on the right.
- **Header:** Minimalist. Breadcrumbs in Arabic + Profile.
- **Main View (Left):**
    - **Top Stats:** 4 Elegant cards. Title (small, muted), Value (Large, Bold), Progress indicator (Subtle small green/red text).
    - **Orders Chart:** Clean line chart with one single blue line. No grid lines (minimalist).
    - **Activity Table:** Ghost-style (no background row colors). Subtle 1px dividers. Badges should be minimalist (small dot + text).

---
**Goal: Create a design that feels like it was designed by a single world-class interface designer with a obsession for consistency.**
