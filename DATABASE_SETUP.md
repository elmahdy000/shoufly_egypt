# 🗄️ تهيئة قاعدة البيانات

تم إنشاء نظام Admin Dashboard احترافي. الآن تحتاج إلى تهيئة قاعدة البيانات بـ sample data.

## الخطوات:

### 1️⃣ تشغيل الـ Migration (إنشاء الجداول)
```bash
npm run db:migrate
```

### 2️⃣ تعبئة البيانات الأولية
```bash
npm run db:seed
```

### أو استخدم الـ Setup Script
```bash
npm run setup:db
```

## ✨ بديل: استخدام الـ API

إذا كنت في بيئة v0 أو لا تستطيع تشغيل CLI commands:

**زر هذا الرابط في المتصفح:**
```
GET /api/admin/init-db
```

سيقوم تلقائياً بـ:
- إنشاء الجداول إذا لم تكن موجودة
- إنشاء بيانات اختبار شاملة (admin, clients, vendors, requests)
- تعبئة قاعدة البيانات بـ 8+ requests تجريبية
